# MinMaxHeap


This program requires to a list of list of natural numbers separate by space  
Use keyboard to choose different options


## Options:
1. Inseret, is ask for a natural number insert to the heap
2. Display, is print out all the elements in the heap in levels
3. Find minimum, is display the minimun element in the heap
4. Find maximum, is display the maximun element in the heap
5. Delete minimum, is delete and display the minimun element in the heap
6. Delete maximum, is delete and display the maximun element in the heap
7. Quit, is quit the program

## Note:
1. The input will stop when enter '0' in the default heap element
2. This program will loop forever until choose option 7 to quit
