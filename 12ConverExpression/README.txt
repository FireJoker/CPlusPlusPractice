# Conver Expression


- This program requires to input prefix, infix or postfix expression
- Detected the notation used for input expression
- Check possible errors for the input expression
- Conver the input expression to postfix, if not its case
- Constructing an expression tree and evaluate
- Output in prefix, infix and postfix notation

## Notes:
1. Handling numbers with more than 1 digit
2. Should leave at least 1 space between operators, operands and symbols
